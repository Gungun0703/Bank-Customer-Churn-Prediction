# Bank Customer Churn Prediction

## Project Overview
This project predicts whether a customer is likely to leave a bank using Machine Learning techniques.

## Objective
Identify customers at risk of churn and determine the factors influencing customer retention.

## Dataset
Bank Customer Churn Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Building
5. Model Evaluation
6. Feature Importance Analysis

## Results
- Logistic Regression model trained successfully.
- Key churn drivers identified.
- Model evaluated using Accuracy, Precision, Recall and Confusion Matrix.

## Key Factors Affecting Churn
- Age
- Balance
- Number of Products
- Active Membership Status
- Geography
